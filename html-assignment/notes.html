Header: < header > < /header > This tag identifies an item as a header.
Body: < body > < /body > Anything with the body is the content of the website.
Superscript: < sup >< /sup > This tag allows you to write in superscript. Example: hello world!
Subscript: < sub > < /sub > This tag allows you to write in subscript. Exampple: hello world!
